# AGENTS.md

1. **Think Before Coding**: State assumptions explicitly, surface ambiguity instead of guessing, and ask when the request is unclear.
2. **Simplicity First**: Implement the minimum code that solves the task; avoid speculative features, abstractions, and configuration.
3. **Surgical Changes**: Touch only code required for the request, match existing style, and clean up only issues introduced by your own change.
4. **Goal-Driven Execution**: Define clear success criteria, verify the result with tests or checks, and iterate until the goal is met.
